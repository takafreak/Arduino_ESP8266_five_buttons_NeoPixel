# Arduino_ESP8266_five_buttons_NeoPixel

Used registers (10k立 x 5, 22k立 x 1) to detect different levels of signals on each mechanical push buttons (red, blue, yellow, green and white) and lit up NeoPixel in corresponding color.


| Button  | Signal range  |
| ------- | ------------- |
| RED     | 450 ~ 520     |
| BLUE    | 520 ~ 590     |
| YELLOW  | 590 ~ 710     |
| GREEN   | 710 ~ 860     |
| WHITE   | 860 ~         |

# Devices

- Adafruit Huzzah Feather ESP8266 x 1
- Adafruit NeoPixel strip x 1
- 10k立 register x 5
- 22k立 register x 1
- Mechanical push button x 5
- LiPo battery or USB power
